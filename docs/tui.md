# TUI Guide

## Overview
The TUI provides a chat-like interface for interacting with your vault.

## Usage
Run:
```sh
python -m mnemosyne.frontends.tui
```

- Use arrow keys to navigate
- Type questions or commands
- See responses in real time

## Features
- Chat with your notes
- Search and create notes
- Discover related content
